These types represent numbers
int --> -1, 0, 2, 3, etc. => integer [-inf, +inf]
float --> 1.1, 1.23, -1.4, 0.0, 3.57, etc
complex --> a +- ib

In python, there is no limit for the size of an int data type . It can store very large integer numbers conveniently
In Python, int does not have a fixed size like C/C++. Python integers can grow as large as memory allows || system allows.
